Orienté objet à Prototype

Un objet

But => Créer un ensemble lié, sous la forme d'attribut avec une valeur

En JS, les objets sont dynamique

    // Création d'un objet (Sans prototype)
    const house = {
        nbRoom: 42,
        color: 'hot pink'
    };

    // Modification d'un attribut
    house.nbRoom = 4;

    // Ajout d'un attribut
    house.hasGarden = true;

Demo d'un prototype

Création d'un model qui permet de créer des objets.
Définir l'ensemble des régles de l'objet (propriétés ? méthodes ?)
Un prototype → C'est une fonction 

    function Person(firstname, lastname) {
        this.firstname = firstname;
        this.lastname = lastname;
    }
    Person.prototype.getFullname = function() {
        return this.firstname + ' ' + this.lastname;
    }

Demo d'une classe

Ce n'est que du sucre syntaxique

    class Person {
        constructor(firstname, lastname) {
            this.firstname = firstname;
            this.lastname = lastname;
        }
    
        getFullname() {
            return this.firstname + ' ' + this.lastname;
        }
    }
