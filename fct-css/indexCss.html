width et height

  • clamp()

    clamp(MIN, VAL, MAX) équivaut à max(MIN, min(VAL, MAX)))

    La fonction clamp() calcule une valeur tout en la limitant entre deux valeurs extrêmes.

    width: clamp(200px, 50%, 100% - 200px);

  • calc()

    calc(expression)

    Opérations simples concernant les longueurs et tailles

    width : calc(100% - 1em);
    calc((100px - 10px - 200px) * 2);
    left : calc(100% / 2 - 2 * 1em - var(--taille) * 1px);

  • min()

    min(val-min,...,val-max)

    width: min(75ch, 50%);

  • max()

    max(val-max,...,val-min)

    padding: var(--blockPadding) max(2rem, 50vw - var(--contentWidth) / 2);
    width: max(45ch, 50%);
    font-size: max(min(0.5vw, 0.5em), 1rem);

GRID

  • minmax()

    minmax(min, max)

    La fonction minmax() définit une dimension comprise entre deux valeurs, pour le dimensionnement des colonnes et des lignes d'une grille.

    grid-auto-columns : minmax(min-content, max-content)
    grid-auto-columns : minmax(100px, 500px)
    grid-template-columns: minmax(100px, 200px) 1fr 1fr;

  • repeat()

    repeat()

    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr))

  • fit-content()

    fit-content(8ch)

    grid-template-columns: fit-content(8ch) fit-content(8ch) 1fr;
