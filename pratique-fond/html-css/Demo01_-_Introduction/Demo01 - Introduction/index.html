Pour créez une document HTML :

  1. Créer un fichier avec l'extension HTML
  2. Choisir une méthode de création :
    1. Manuellement :
      1. Définir le DOCTYPE
      2. Placer la balise HTML
      3. Placer la balise head
      4. Placer la balise body
    2. Automatique : écrire "!" dans VSCode

Vocabulaire :

Balise
Balise est un élément HTML permettant de stocker ou structurer une information.
Attribut
Paire d'information ajouté dans une balise.
