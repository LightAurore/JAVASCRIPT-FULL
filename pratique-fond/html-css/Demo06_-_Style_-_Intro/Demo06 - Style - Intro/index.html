Démonstration 06 : Page avec style

Ce document ne contient aucun code de style, pourtant il a une mise en page!

Il est lié à un document de type CSS (Cascade StyleSheet)

Il est possible de malgré tout ajouter du style grâce à la balise "style"

Avantages :

  • Code séparé, nous ne mélangeons pas l'HTML et le CSS
  • Le CSS peut-être ré-utilisé dans plusieurs pages HTML
  • Séparation conventionnel du contenu et du design

Comment manipuler le style CSS

  1. On ajoute une balise link à notre head
  2. On détermine les sélecteurs de notre css
  3. On définit les propriétés du sélecteur
