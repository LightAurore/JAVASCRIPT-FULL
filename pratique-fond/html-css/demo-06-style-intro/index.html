Démonstration 06 : Page avec style

Ce document ne contient aucun code de style, pourtant il a une mise en page!

il est lié à un document de style CSS (Cascade StyleSheet)

Il est possible malgré tout d'ajouter du style grâce à la balise "style" .

Avantages

  • Code sépar", nous ne mélangeons pas l'HTML et le CSS
  • Le CSS oeut-être ré-utilisé dans plusieurs pages HTML
  • Séparation conventionnel du contenu et du design

Comment manipuler le style CSS

  1. On ajoute une balise link à notre head
  2. On détermine le selecteur de notre CSS
  3. On définit les propriétés du sélecteur
